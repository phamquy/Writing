# A Swift Tour

Theo thông lệ, chương trình đầu tiên được viết khi học ngôn ngữ mới nên in chữ "Hello, World!" ra màn hình. Với Swift, có thể viết đơn giản như sau

> `print ("Hello, World!")`

Lập trình viên với C hoác ObjectiveC, thì sẽ thấy cú pháp này quen thuộc. Tuy nhiên trong Swift, một dòng lênh như trên thực chất cũng là một chương trình hoàn chỉnh. Ở đây không cần phai import một thư viện ngoai cho viêc input/output hoác xử lí chuỗi kí tự. Code được viết ở mức global đưọc dùng như một chương trình, tức là không cần phải có hàm `main()`. Dấu `;` không cần thiết khi nằm ở cuối dòng code.
